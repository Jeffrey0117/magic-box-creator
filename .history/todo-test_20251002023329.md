## 🚨 已知風險與應對

### 風險 1：Supabase 免費額度
- **限制**：500MB / 500 萬次 API 請求
- **監控**：Supabase Dashboard 每週檢查
- **應對**：達 80% 時考慮付費或限制功能

### 風險 2：惡意濫用
- **風險**：大量建立資料包或解鎖
- **監控**：觀察異常流量
- **應對**：實作 Rate Limiting（V8）

### 風險 3：內容違規
- **風險**：創作者上傳不當內容
- **監控**：用戶檢舉機制（V8）
- **應對**：使用條款 + 免責聲明
